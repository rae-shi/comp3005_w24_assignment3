1.  Steps to run and compile the application
    a. Create a database named 'comp_3005_a3' in PostgreSQL. Make sure it's connected.
    b. Open the Query Tool under the Tools, click 'Open File' button, import the script file named 'script.sql' under the 'comp3005_w24_assignment3_database_scripts' folder
    c. Click the 'Run' button and it will create and initialize a table named students
    d. Import the 'comp3005_w24_assignment3_application_code' folder inside IntelliJ
    e. Open the main folder under the src folder. Then Open the Main class under the java folder
    f. Click the Run button on the top right side
    g. The application should run successfully
    h. To check each fucntion, please comment out certain code inside the main method 
    i. To check the data in PostgreSQL, plese run Query 'SELECT * FROM students' inside the 'comp_3005_a3' database
